# Python-Shopping-Assistant

My first Python project, created while learning Python fundamentals.

Description: This program simulates a simple shopping assistant.
It allows users to:
Enter a nickname
Add grocery items
Input the price of each item
Calculate the total cost
Display a summary of purchased items
Features
Nickname validation
Currency formatting with commas and decimal places
Multiple item support
Running total calculation
Receipt-style output

Concepts Practiced:

Variables
User input (input())
Conditional statements (if, elif, else)
String methods (isalpha(), replace(), capitalize())
Type conversion (float())
f-strings and format specifiers
Basic debugging

Example Output:
Enter your nickname: Terrence

What is item 1?: Rice
What is the price of Rice?: 280.50

...

Your total is 1,377.60

Thank you for shopping with us Terrence!

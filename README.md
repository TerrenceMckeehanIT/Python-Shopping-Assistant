# Python-Shopping-Assistant
A beginner Python project created while learning Python.

Features:
Accepts a user's nickname
Validates nickname input
Allows users to enter grocery items and prices
Calculates the total cost of all items
Formats the total with commas and two decimal places
Prints a receipt-like summary of purchased items

Concepts Used:
Variables
User Input (input())
Conditional Statements (if, elif, else)
String Methods
Type Conversion (float())
f-Strings
Basic Input Validation

Example Output:
